## Project Description
This is a React-based Student Portal Dashboard built using:
- React (CDN)
- Tailwind CSS
- Mock backend data

The project demonstrates component-based UI, state management, search functionality, and responsive design.

## Features
- Student listing dashboard
- Profile detail page
- Analytics dashboard
- Settings page
- Search functionality

## How to Run
1. Clone or download the repository
2. Open `index.html` in any modern browser
3. No backend or build setup required

## Note
React and Tailwind are loaded via CDN for simplicity and fast setup.
